Static content

  • Spring Boot static content function

How to create static content

  • Spring
    src/main/resources/static/hello-static.html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>static content</title>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
</head>
<body>
This is static content
</body>
</html>

Execution

  • http://localhost:8080/hello-static.html

MVC and template engine

  • MVC: Model, View, Controller Controller
    src/main/java/hello.hellospring/controller/HelloController.java
@Controller
public class HelloController{
    @GetMapping("hello-mvc")
    public String helloMvc(@RequestParam("name") String name, Model model) {
        model.addAttribute("name", name);
        return "hello-template";
    }
}

View
src/main/resoucres/template/hello-template.html

<html xmlns:th="http://www.thymeleaf.org">
<body>
<p th:text="'hello ' + ${name}">hello! empty</p>
</body>
</html>

Execution

  • http://localhost:8080/hello-mvc?name=spring

API

@ResponseBody returns value
src/main/java/hello.hellospring/controller/HelloController.java

@Controller
public class HelloController {
    @GetMapping("hello-string")
    @ResponseBody
    public String helloString(@RequestParam("name") String name) {
        return "hello " + name;
    }
}
  • If you use @ResponseBody, viewResolver will not be used
  • Instead HTTP BODY’s content is returned(Not HTML BODY TAG)

Execution

  • http://localhost:8080/hello-string?name=spring @ResponseBody returns object
    src/main/java/hello.hellospring/controller/HelloController.java
@Controller
public class HelloController {
    @GetMapping("hello-api")
    @ResponseBody
    public Hello helloApi(@RequestParam("name") String name) {
        Hello hello = new Hello();
        hello.setName(name);
        return hello;
    }

    static class Hello {
        private String name;

        public String getName() {
            return name;
        }

        public void setName(String name) {
            this.name = name;
        }
    }
}
  • @ResponseBody is used, the object is returned in JSON format

Execution

  • http://localhost:8080/hello-api?name=spring

Use principle of @ResponseBody

  • HTTP BODY’s value is directly returned
  • Instead of viewResolver, HttpMessageConverter is in action
  • Basic string process: StringHttpMessageConverter
  • Basic object process: MappingJackson2HttpMessageConverter
  • byte process and others use HttpMessageConverter
