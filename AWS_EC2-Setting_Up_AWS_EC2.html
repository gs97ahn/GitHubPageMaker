AWS EC2 - Setting Up AWS EC2

What is AWS EC2

In order to have a backend application. You need to have a server. Not everyone has a server computer in their homeüòÖ. This is when you use AWS EC2. EC2 stands for Elastic Compute Cloud. You can borrow a server computer with a specific pricing that AWS provide. Basically, you can understand it as a rental server.

Step 4Ô∏è‚É£

Select the instance type that you would like. Depending on the computing power that you need, this may vary. Of course, if you choose a powerful instance, you would need to pay more for the usage.

Step 5Ô∏è‚É£

Set up a key pair. This is not required, however it is important that you need to set this up for security reasons as you do not want anyone else to access your ec2 instance. Click on the ‚ÄúCreate new key pair‚Äù if you do not have a key pair already.

After clicking on the ‚ÄúCreate new key pair‚Äù, name your key pair and select the options that you need.

You should probably stick with RSA because ED25519 is not supported for Windows instance in case you are need Windows instance.

For private key file format, depending on your favor, whether it is OpenSSH or PuTTy, select the option.

Finally, click Create key pair and keep the file somewhere secure. If you lose this file, you will not be able to get into your ec2 instance ever again.

For the sake of my example, I am going to select RSA and .pem format.

Next is setting up the network settings. If you have a constant IP address, and you will want to access the EC2 instance only from the specific computer, Click on the Anywhere 0.0.0.0/0 and customize it or select ‚ÄúMy IP‚Äù which would be your current IP address.

If you are having a backend application in the EC2, you need to allow HTTPs or HTTP or both traffic from the internet. This is your clients accessing your server for requests and responses.

For my example, I am going to run a Spring Boot application, so I am going to allow HTTP traffic only. Also, I am going to allow SSH traffic from Anywhere.

Step 7Ô∏è‚É£

Next step is setting up storage. For free tier, AWS allows you to have upto 30GB, but you probably would like to check on the pricing to reduce surprises that you might encounter in the future.

For my example, I am going to give 10GiB to my storage.

Step 8Ô∏è‚É£

If you want your instance to have GPU or other advanced setting, you can configure it in ‚ÄúAdvanced details‚Äù.

For my example, I do need any of the advanced settings for now, so I am just going to ignore this part.
