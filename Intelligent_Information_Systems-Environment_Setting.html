Data analysis in computer

  • For data analysis use Python or R
    • Python is high level programming language that was published in 1997 by Gudio van Raossum. He named it python because of his favorite comedy TV show Monty Python’s Flying Circus.

Tools used for data analysis

  • Anaconda
    • Named anaconda because there are many projects or packages that are related to snakes.
  • Jupyter notebook
    • Open source web application
    • Run partial codes and check the output
    • Named jupyter because of three main languages Julia, Python, and R.

Anaconda commands

Create virtual environment

Get a list of all my environments

conda env list

Create a new environment named py35, install Python 3.5

conda create --name py35 python=3.5

Active a existing environment named py35

conda activate py35

Delete an environment named py35 and everything in it

conda env remove --name py35

Jupyter notebook commands

Run jupyter notebook

jupyter notebook
